# 제 2절 국어 문법의 이해
 
 ### 1. 음운의 개념
   - 음운 : 말의 뜻을 구별해주는 소리의 최소 단위 (추상적, 관념적) 
     - 음운은 음성을 구별하는 소리 인식 장치
     - 한국어는 40개, (V B 구별 못함)

  - 분절 음운(음소) = 자음(19) + 모음(21)
  - 비분절 음운(운소) = 소리 길이, 억양...

- **최소대립쌍** 
  - 의미를 구별하게 하는 음운을 가진 단어 쌍
  - 오직 한가지 요소에 의해서만 의미 구별

   >님 - 김 (ㄴ ㄱ)   
   >님 - 남 (ㅣ ㅏ) 

  - 음운 = 음소 + 운소

 ### 2. 음절의 개념과 형성
 - 음절 : 실제 발음나는 소리
 - 소리를 내려면 모음이 필요 (자음 단독으로 X)
 - '아' : ㅇ는 소리가 없음
 
 - ex
   > 아 : 모음    
   > 안 : 모음 + 자음   
   > 값 : 자음 + 모음 + 자음
